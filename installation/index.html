    Installation

    Complete installation guide for AC Bridge.

    System Requirements

    • OS: Windows 10/11 (64-bit)
    • Python: 3.11 or higher
    • Game: Assetto Corsa (original, not Competizione)
    • Memory: 2GB+ RAM
    • Disk: 500MB for dependencies

    Step 1: Install Python

    Option A: Microsoft Store (Recommended)

    # Search "Python 3.11" in Microsoft Store and install

    Option B: python.org

    Download from python.org/downloads

    Verify installation:

    python --version
# Should show Python 3.11.x or higher

    Step 2: Install uv (Optional but Recommended)

    uv is a fast Python package installer:

    # PowerShell (as Administrator)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

    Restart your terminal, then verify:

    uv --version

    Step 3: Install vJoy

    vJoy is required for sending control inputs to AC.

    1. Download vJoy 2.1.9
    2. Run installer
    3. Configure device 1:
    4. Open "Configure vJoy" from Start Menu
    5. Select "Device 1"
    6. Enable axes: X, Y, Z, RZ
    7. Enable 12 buttons
    8. Click "Apply"

    Verify in Windows Device Manager: - Devices and Printers â†’ vJoy Device should appear

    Step 4: Install AC Bridge

    Clone Repository

    git clone https://github.com/wongd-hub/assetto-corsa-gym.git
cd assetto-corsa-gym

    Install Dependencies

    With uv (recommended):

    uv sync
uv pip install -e .

    With pip:

    pip install -e .

    Step 5: Verify Installation

    Test Telemetry

    Start Assetto Corsa, enter a session, then:

    uv run ac-bridge test-telemetry --hz 10

    You should see live telemetry output:

    Speed: 145 km/h | RPM: 5420 | Gear: 3 | Throttle: 85%

    Test vJoy Control

    uv run ac-bridge test-control

    Interactive menu should appear. Test each axis to verify vJoy is working.

    Run Smoke Test

    Combined test of telemetry + control:

    uv run ac-bridge smoke-test

    Step 6: Configure AC Controls

    Map vJoy in Assetto Corsa:

    1. Launch AC â†’ Options â†’ Controls
    2. Change device to "vJoy Device"
    3. Map axes:
    4. Steering â†’ vJoy X axis (Axle 1)
    5. Throttle â†’ vJoy Y axis (Axle 2)
    6. Brake â†’ vJoy Z axis (Axle 3)

    7. Clutch â†’ vJoy RZ axis (Axle 6, optional)

    8. Map buttons (recommended):

    vJoy Button Function Required for AC Bridge
    Button 1 Next gear No
    Button 2 Previous gear No
    Button 3 Brake balance â†’ front No
    Button 4 Brake balance â†’ rear No
    Button 5 Traction control + No
    Button 6 Traction control - No
    Button 7 Restart race Yes (for reset)
    Button 8 Restart session No
    Button 9 Start race Yes (for reset)

    Required Buttons

    Buttons 7 and 9 must be mapped for bridge.reset() to work. The reset sequence presses button 7 (restart race), waits 2s, then presses button 9 (start race).

    1. Save and test - drive around to verify controls work

    Troubleshooting

    Python not found

    # Check Python is in PATH
python --version

# If not found, add to PATH:
# Settings â†’ System â†’ About â†’ Advanced system settings
# â†’ Environment Variables â†’ Path â†’ Add Python install dir

    uv not recognized

    # Restart terminal after installing uv
# If still not working, add to PATH:
%USERPROFILE%\.cargo\bin

    vJoy device not found

    # Check if vJoy is installed
import pyvjoy
device = pyvjoy.VJoyDevice(1)  # Should not raise error

    If error: 1. Reinstall vJoy 2. Configure device 1 in vJoy Configure tool 3. Reboot Windows

    ModuleNotFoundError

    # Make sure you installed in editable mode
cd assetto-corsa-gym
uv pip install -e .

# Or with pip
pip install -e .

    AC telemetry not available

    Make sure: 1. AC is running (not just launcher) 2. You're in a session (practice/race, not main menu) 3. Shared memory is enabled (usually enabled by default)

    vJoy controls not working in AC

    1. Check device in Windows Device Manager
    2. Test vJoy with uv run ac-bridge test-control
    3. Re-map controls in AC (sometimes AC resets mappings)
    4. Make sure no other controller is active in AC

    Optional: Development Setup

    For contributing or modifying AC Bridge:

    # Install dev dependencies
uv sync --dev

# Install pre-commit hooks (optional)
pip install pre-commit
pre-commit install

# Run tests
pytest

# Build docs locally
uv run mkdocs serve
